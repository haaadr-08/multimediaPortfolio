Programmer Documentation

Technical information for updating the Historical Events app.

Modifying the Dataset

All event information is stored inside data.json. Each event object contains a title, year, description, image path, and audio path.

Example JSON Object

{
    "title": "World War I",
    "year": "1914-1918",
    "image": "images/ww1.jpg",
    "description": "Historical event information here.",
    "audio": "audio/ww1.mp3"
}

Adding New Events

  1. Add a new event object inside data.json.
  2. Add the new image inside the images folder.
  3. Add the new MP3 file inside the audio folder.
  4. Use the correct file paths in data.json.
  5. Save the files and refresh the browser.

Project Files

  • index.html - main app page
  • style.css - page design
  • script.js - loads JSON data
  • data.json - stores event content
  • manifest.json - stores app install settings
  • service-worker.js - service worker file
