Historical Events Progressive Web App

A learning app about important events in world history.

About This App

Historical Events is a Progressive Web App that teaches users about major historical events using images, descriptions, years, and audio narration. The app can be installed and used like a regular application.

Features

  • Historical event cards
  • Images for each event
  • Audio narration
  • Installable PWA
  • Offline support
  • Responsive layout

How to Install

  1. Open the app in Chrome.
  2. Click the Install App button.
  3. Click Install when the popup appears.

How to Use

  1. Open the app.
  2. Scroll through the historical event cards.
  3. Read the event descriptions and years.
  4. Press the audio controls to listen to the narration.
