Historical Events Documentation

Project documentation for the Historical Events PWA.

Project Overview

Historical Events is an educational web app that teaches users about important historical events. The app uses event cards with the event name, year, image, description, and audio narration.

What Was Completed

  • Created 8 historical event cards
  • Added matching images for each event
  • Added real MP3 audio narration for each event
  • Used JSON to store event information
  • Created an app icon and manifest file
  • Added an Install App button
  • Created promotional, documentation, and programmer documentation pages

Files Used

  • index.html - main app page
  • style.css - design and layout
  • script.js - loads event cards from JSON
  • data.json - stores historical event data
  • manifest.json - app install information
  • service-worker.js - service worker file
  • promo.html - promotional page
  • docs.html - project documentation
  • programmer.html - programmer documentation

Technologies Used

  • HTML5
  • CSS3
  • JavaScript
  • JSON
  • Web App Manifest
  • MP3 audio files

How Audio Works

The narration audio files were created with text-to-speech tools. Each MP3 file is stored in the audio folder and connected to the correct historical event inside data.json.
